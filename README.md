# Guess the Music Game

Aplicacao React + Vite para um jogo de adivinhar musicas.

## Stack inicial

- React
- Vite
- Axios
- CSS responsivo

## Como rodar

```bash
npm install
npm run dev
```

O projeto roda por padrao em `http://localhost:5173`.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## Deezer API

O client HTTP inicial esta em `src/services/deezerClient.js`.

Para configurar a URL base, crie um `.env` a partir de `.env.example`:

```bash
VITE_DEEZER_API_URL=https://api.deezer.com
```
